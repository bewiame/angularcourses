### Setup

* Install Protractor

    `npm install --save-dev protractor`
* Make sure your development server is running
* Run Selenium if needed:

    `node node_modules/.bin/webdriver-manager start`
* Run Protractor:

    `node node_modules/.bin/protractor protractor.conf.js`
